# ml-examples-w3schools

Machine learning code examples from the w3schools tutorial "[Learn Machine Learning](https://www.w3schools.com/python/python_ml_getting_started.asp)".

## Python Requirements

Uses Python 3 (at least 3.6)

Required packages:

* NumPy
* SciPy
* Matplotlib
* Scikit-learn
* PyDotPlus
